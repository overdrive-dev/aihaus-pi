# aih-update

Update aihaus-pi package resources and report aihaus-pi status. In target repositories, do not run the global Pi runtime update unless `--with-pi` is explicit. Use `/aih-update status` for read-only status. Do not repair corrupted local state; recommend `/aih-repair` for that.
